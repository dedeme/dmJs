Usar el menú

| Nuevo

Cuando la luz en la esquina superior-derecha está encencida (amarillo), es posible hacer 'click' sobre el icono y crear un nuevo sudoku. El sudoku previo será descartado.

Si la luz está apagada es preciso esperar hasta que se encienda.

Copiar

Este botón permite copiar un sudoku externo (de un periódico, por ejemplo)

Recuperar

Permite recuperar un sudoku previamente guardado con 'Guardar'

Guardar

Permite guardar el sudoku actual. Es posible guardar hasta nueve sudokus diferentes.

Cuando se guarda una nueva versión de un sudoku en curso, esta nueva versión sobreescribe la anterior. Esto es, sólo es posible tener una versión de cada sudoku.

Los sudokus guardados pueden ser restaurados con 'Recuperar'.

| Cambiar el lenguaje

Haciendo 'click' sobre el texto, se cambia el lenguar de la aplicación.

Cambiar el nivel

Es posible subir y bajar la dificultad del juego haciendo 'click' sobre los signos mas y menos.

Bolígrafo - lápiz

Este botón permite cambiar el modo de escritura. Es posible escribir con bolígrafo o con lápiz.

Si se escribe con bolígrafo, los números son mostrados en negro. Estos números no pueden ser borrados con la brocha gris (ver a continuación)

En caso de escribir con lápiz, los numeros serán mostrados en gris. Esto permite user la brocha gris para borrarlos.

| Borrar

Este icono está en gris cuando se selecciona el sistema de escritura 'con lápiz'.

Cuando el icono está gris y se hace 'click' sobre él, solo se borrarán los números grises. En otro caso todos los números serán borrados, esto es, se deberá empezar el sudoku de nuevo.

Mostrar errores

Es posible verificar si se han escrito números incorrectos haciendo 'click' sobre este icono. Los números incorrectos aparecerán marcados en rojo. Si se vuelve a pulsar sobre el bótón el sudoku volverá a su estado normal.

Resolver

Puedes engañarte a ti mismo haciendo click aquí... o simplemente tratar de verificar si el sudoku tiene una solución válida. En cualquier caso, este icono muestra el sudoku resuelto.

 

Escribir números

Es posible escribir desde el teclado, con el ratón o a través de una pantalla táctil.

Desde el teclado se puede:

  • Para escribir números: Usar la zona principal o el bloque númerico.
  • Para borrar números: Usar las teclas 'espacio', 'suprimir' o 'retroceso'.
  • Para moverse: Las teclas del cursor.

Con el ratón o a través de una pantalla táctil es preciso utilizar el panel numérico que viene en la parte inferior de la ventana.

  • Escribir números: Haciendo 'click' en el panel numérico.
  • Borrar números: Haciendo 'click' en la tecla en blanco del panel numérico.
  • Moverse: Haciendo 'click' directamente en el tablero del sudoku.
